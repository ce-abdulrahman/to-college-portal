1. Location & Navigation Tools

🔍 Search bar بۆ گەڕان بە ناوی پارێزگا، زانکۆ، یان بەش

🧭 Locate user (Current Location) بۆ نیشاندانی شوێنی ئێستا

🗺️ Zoom-to-extent for all layers (کرتەیەک بۆ گەڕانەوە بۆ نیشانەی سەرەکی)

🔄 Layer control بۆ هەڵبژاردنی ڕووبەرەکانی پارێزگا، زانکۆ، کۆلێژ، بەش بە جیا


🎨 2. Style & Visualization

🌈 Color by type (مثلاً بەشە زانستی بە شین، وێژەیی بە سوور)

🗾 Dynamic icon بۆ زانکۆ و کۆلێژ (وێنە یان logo)

🖼️ Cluster markers ئەگەر ژمارەی پۆینت زۆرە، بۆ ئاسانکاری نیشاندان

✨ Hover tooltip (ناوی پۆینتەکان لەسەر hover نیشان بدرێت)

🔲 Custom basemap switcher بۆ گۆڕینی بینینی نەخشە (Satellite, Terrain, Dark)


🧩 3. Info & Data Display

🗒️ Advanced popup (بە دوو تبّ: Info + Scores + Gallery)

📊 Mini charts لە popup ـەکان (local vs external score bar)

🧾 Details sidebar — کلیک لە هەر پۆینتێک، لە تەنیشتەوە ناو زانیارییەکەی بکەوە بە تەواوی

🧮 Statistics summary (هەژمارەی بەشەکان، نمرەی گشتی، پۆلیگۆنەکان...)


⚙️ 4. Interactivity & Filtering

🧩 Filter panel بۆ:

پارێزگا

جۆری بەش (زانستی، وێژەیی، تێکەڵ)

نمرەی local/external بەپێی range slider

sex (پیاو/ژن/هاوپەیک)

🎛️ Checkbox layers بۆ گۆڕینی visible بودنەکانی پارێزگا، زانکۆ، کۆلێژ

📍 Clickable breadcrumbs بۆ گەڕانەوە (پارێزگا → زانکۆ → کۆلێژ → بەش)


🛰️ 5. Extra Mapping Tools

🖌️ Draw tool (Polygon/Marker/Line بۆ پێوەندی خوێندنی نوێ)

📏 Measure tool (مسافت/ڕووبەرەکە پێوەری)

💾 Export tool (بیرکردنەوەی GeoJSON بۆ پەیوەندیدان بە ArcGIS/QGIS)

🕹️ Mini map overview (نەخشەی بچووک لە کونجی خوارەوەی چەپ)

🌍 Print / Save map view بۆ PDF یان PNG

🧠 6. Smart & Dynamic Features

⚡ Live sync (کاتێک lat/lng یان geojson نوێ بێت لە DB، خۆکارەوە map refresh بکات)

💬 Tooltip info mode (کرتەیەکی خێرا بۆ نیشاندانی ناوەکان بێ popup)

📍 Highlight animation کاتێک کلیک دەکەیت سەر پۆینت، بە شێوەی نیشاندەر دەردەکەوێت

🌐 Language switcher (کوردی / ئینگلیزی)

💡 پێشنیاز بۆ قۆناغی داهاتوو

🧭 Routing path: پێویست بە leaflet-routing-machine (ڕێگای گەیشتن بە زانکۆ/کۆلێژ)

📆 Year filter: ئەگەر داتاکان بە ساڵ جیاوازن، بۆ گۆڕینی ساڵی data.

📈 Charts page: گرافی گشتی نمرەکان بە شێوەی bar/pie chart.

💬 Comments / Notes system: بەکارهێنەر بتوانێت نوت بنووسێت لەسەر پۆینت.
